About Project

This is Pokémon image classification web application, using pretrained-model from Trolukovich's kaggle notebook. Model was trained on
 Pokémon Generation One dataset. At the moment, model are only capable of predicting 5 classes being Mewtwo, Pikachu, Charmander, Bulbasaur and Squirtle
